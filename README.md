# fx2preset-lite
內部使用工具

## 還原文件
```
fx2preset-lite restore
```

## 列出所有preset
```
fx2preset-lite list
```

## 打開preset目錄
```
fx2preset-lite find "American Dream"
```

## 登出tonecloud
```
fx2preset-lite logout
```

## 切換tonecloud環境
### prodction
```
fx2preset-lite env prod
```

### staging
```
fx2preset-lite env stage
```
